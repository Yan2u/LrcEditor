## LRCEditor By Wyx

![logo](https://s2.ax1x.com/2020/02/23/33EXL9.png)

一个简单的歌词编辑器，带有歌词编辑，歌词搜索，歌词翻译的功能

在听歌的同时加入歌词和编辑歌词，同时对歌词进行调整

### Usage

程序 `LrcEditor` 位于 `LrcEditor\bin\Debug` 里，也可以在 `LrcEditor\publish` 里面找到 `setup.exe` 进行安装
